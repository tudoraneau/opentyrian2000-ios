OpenTyrian Engine Unofficial

This app is a mobile port of the OpenTyrian 2000 open-source game engine.

This is an unofficial project and is not affiliated with the original developers of Tyrian, OpenTyrian, or OpenTyrian2000 project.

Important Notice

This app is a game engine and does not include any game content or assets. To use it, you must provide your own compatible data files.

The full gameplay experience is available for free once you supply your own game data.

Features

  • Smooth performance on modern iOS devices
  • Portrait and landscape mode support
  • Touch controls optimized for mobile play
  • Bluetooth controller support available via an optional in-app purchase

Important

This app does not include the original Tyrian 2000 game or its assets. Users must legally obtain and import their own compatible data files.

Instructions

  1. Connect your iPhone to a computer
  2. Open the Finder app (macOS) or iTunes (Windows)
  3. Select your device and navigate to the Files section
  4. Copy your legally obtained Tyrian 2000 data files into the OpenTyrian Engine Unofficial folder
  5. Relaunch the game
